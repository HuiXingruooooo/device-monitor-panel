💡 下载提示

1. 完整项目源码包含所有模块的源代码和库文件，推荐使用 Arduino IDE 或 VS Code 打开编辑。
2. 固件烧录包是已经编译好的二进制文件，可直接使用 ESP32 Download Tool 烧录到设备。
3. 首次使用请确保安装 USB 转串口驱动，否则可能无法识别设备。
4. 如需修改 WiFi 配置，请在源码中修改 WIFI_SSID 和 WIFI_PASSWORD 宏定义。
5. MQTT 服务器地址和凭证可在代码中修改，或烧录后通过串口命令配置。
