产品介绍

PCT_100 化工控制实训装置 是一款专为化工自动化教学设计的高性能智能控制系统。本装置采用先进的 ESP32 微控制器作为核心，集成了多种工业级传感器和通信模块，能够模拟真实的化工生产过程控制场景。

设备型号：PCT_100 | 软件版本：V1.2 | 设备 ID：PCT_100_043 | 开发日期：2026-05-29

主要应用场景：

  • 化工自动化控制实训教学
  • 工业物联网（IIoT）技术演示
  • 传感器数据采集与监控
  • 智能家居/实验室环境监控
  • MQTT 物联网通信学习

本装置采用模块化设计理念，各功能模块独立工作又协同配合，支持 WiFi 无线连接和 MQTT 物联网协议，可轻松接入云平台实现远程监控和数据分析。配备 0.96 寸 OLED 显示屏实时显示传感器数据和系统状态，独立按键支持本地参数设置和手动控制，满足教学实训的多样化需求。

功能特性

🌡️ 温度监测

DS18B20 数字温度传感器，测量范围 -55°C ~ +125°C，精度±0.5°C，阈值可配置（默认 35°C）

💡 光照检测

高精度 ADC 光敏电阻检测，量程 0-1000 Lux，阈值可调（默认 200 Lux），自动联动控制

📶 WiFi 连接

ESP32 内置 WiFi 模块，支持 AP/STA 双模式，配置持久化存储，断线自动重连

🔗 MQTT 通信

支持 MQTT 协议接入云平台，发布/订阅模式，设备唯一 ID 标识，安全可靠

🖥️ OLED 显示

0.96 寸 I2C OLED 屏幕，128×64 分辨率，实时显示传感器数据、网络状态

按键控制

独立按键设计，支持参数设置、模式切换、手动控制，工业级耐用设计

声光报警

LED 指示灯和蜂鸣器双重报警，温度/光照超限自动触发，阈值可配置

⚡ 继电器输出

大功率继电器控制，支持外部设备联动，光照/温度超限自动开关

硬件组成

PCT_100 装置由以下核心模块组成：

  • ESP32 主控模块：系统核心，负责数据处理、网络通信、逻辑控制
  • DS18B20 温度传感器：单总线数字温度传感器，防水封装
  • 光敏检测模块：光敏电阻 + 运放电路，输出模拟信号至 ADC
  • OLED 显示模块：0.96 寸 I2C 接口，显示系统状态和传感器数据
  • 按键模块：独立按键，支持参数设置和手动控制
  • LED 指示模块：状态指示灯，温度/光照超限报警
  • 继电器模块：5V 驱动，控制外部设备电源
  • WiFi 天线：板载 PCB 天线，WiFi 信号收发

注意：继电器输出端可承受最大 10A/250VAC 负载，请勿超限使用。接线时请确保断电操作。

使用说明

1. 上电启动：

  • 连接 5V 电源，系统自动启动
  • OLED 显示启动画面，WiFi 开始连接
  • 连接成功后显示实时传感器数据

2. 参数设置：

  • 短按按键：切换显示页面
  • 长按按键（3 秒）：进入设置模式
  • 设置模式下短按切换参数，长按确认
  • 可设置温度阈值、光照阈值、MQTT 参数等

3. 自动控制模式：

  • 温度超过阈值（默认 35°C）→ LED 报警 + 继电器闭合
  • 光照低于阈值（默认 200 Lux）→ 继电器闭合（开灯）
  • 数据自动上报 MQTT 服务器（间隔 5 秒）

4. 手动控制模式：

  • 通过 MQTT 控制指令远程开关继电器和 LED
  • 本地按键可手动切换继电器状态

维护保养

日常维护：

  • 定期清洁设备表面，避免灰尘堆积影响散热
  • 检查接线端子是否松动，确保连接可靠
  • 避免在潮湿、高温、强磁场环境中使用
  • 长期不用请断开电源，存放在干燥环境

故障排查：

  • 无法连接 WiFi：检查账号密码是否正确，确认路由器正常工作
  • 温度显示异常：检查 DS18B20 接线，确认 4.7kΩ上拉电阻已连接
  • OLED 不显示：检查 I2C 接线（SDA/SCL），确认供电正常
  • 继电器不动作：检查 GPIO5 输出和继电器模块供电

技术支持：如遇无法解决的故障，请联系设备供应商或查看项目源码获取帮助。
